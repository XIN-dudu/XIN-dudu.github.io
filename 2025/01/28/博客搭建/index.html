博客搭建

JiaJiongZhang2025-01-282025-01-29

准备工具

1.nodejs

2.git

3.hexo 在命令行中输入npm install hexo -cil -g

注：nodejs 版本推荐12.22.12

结果类似这样就是下载好了

检查是否安装成功

win+r 输入cmd 打开命令行

依次输入这三行，如果正确显示版本则说明安装成功

搭建github仓库

新建仓库

仓库名填 用户名.github.io

其他按照箭头依次点击即可

生成ssh连接仓库

1.本地操作

桌面新建文件夹

右键空白打开命令行

命令行中输入 ssh-keygen -t rsa -C “邮件地址”

然后敲四次回车

找到生成的文件，文件地址大概是”C:\Users\86158.ssh\id_rsa.pub” 这样

右键用记事本打开，ctrl+A ctrl+C 全选复制文件内容

2.打开github

打开github 点击右上角头像，点击Settings,然后左侧找到SSH and GPG keys

标题可以随便输，内容粘贴已经复制的内容，然后add

3.测试是否连接成功

打开刚刚的文件夹，右键git bush here(没找到点显示更多选项)

打开后输入 ssh -T git@github.com

如果显示

则表示连接成功

本地生成博客内容

新建Blog文件夹 右键git bush here

输入 hexo init(如果‘command not find’，就在前面加上npx，如：npx hexo init)

然后输入 hexo install

接下来输入 hexo g

​ hexo s

显示这个则表示已经在本地跑起来了 可以选中网址右键打开，也可以在浏览器输入上述网址打开（注：如果右键打开不行试一下在浏览器输入打开）如果还不行可能是nodejs版本问题，推荐使用12.22.12

CTRL+C 停止服务

上线博客到github仓库

1.打开github刚刚创建的仓库

复制http 网址

2.打开刚刚创建的Blog文件夹

打开Blog文件夹中的 “_config.yml” 文件

划到最下面，把deploy 后面的内容改成

deploy:

type: git

repository: 此处粘贴刚刚赋值的网址

branch: main

保存退出

3.安装自动部署发布工具

打开Blog文件夹，git bush here

输入 npm install hexo-deployer-git –save

然后关掉再git bush here

再打开git bush here 输入 hexo g (发布)再输入 hexo d (上传)

报错解决

1.如果是第一次使用git 需要先配置一下（git bush输入）

git config –global user.email “你的邮箱”
git config –global user.name “你的名字”

2.如果网络连接出错换自己的手机流量热点试试

结束

到此博客搭建已经结束，可以在浏览器输入 http://“仓库名字” 进行访问。但由于github限制，可能访问会有点慢。

教程有许多不尽美的地方。如果过程中遇到了问题可以自己去网上搜，我也是把我遇到的问题在上面写了解决方法。

最后博主也只是一个小白，其余博客美化等功能暂时不涉及。
